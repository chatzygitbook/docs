| Section | Main Purpose | Key Features |
| :--- | :--- | :--- |
| **Contacts** | Manage individual users | View, edit, import/export, tag, add attributes, blacklist |
| **Lists** | Manual user grouping | Create static groups, export, use for campaign targeting |
| **Segments** | Dynamic user grouping | Create rule-based groups, exportable, auto-updating |
| **Exports** | Export log & file access | Track export activity, download files, see export timestamps |
