These are standard **data types** you can use in the Chatzy Sandbox environment.  
They define the kind of values a variable can hold.


| Type Name | Description |
|-----------|-------------|
| **str**   | String type – stores text values. |
| **int**   | Integer type – stores whole numbers. |
| **float** | Float type – stores decimal numbers. |
| **list**  | List type – ordered collection of items. |
| **dict**  | Dictionary type – key-value mapping of items. |
| **set**   | Set type – unordered collection of unique items. |
| **tuple** | Tuple type – ordered, unchangeable collection of items. |
