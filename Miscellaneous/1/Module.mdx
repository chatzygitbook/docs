These are built-in **modules** available in the Chatzy Sandbox environment.  
They provide ready-made functionality you can use without installing anything extra.


| Module Name    | Description |
|----------------|-------------|
| **httpx**      | Sends HTTP requests from your bot code. |
| **json**       | Parses and formats JSON data. |
| **ElementTree**| Parses XML data. |
| **re**         | Supports searching and pattern matching using regular expressions. |
| **datetime**   | Works with date and time values. |
| **timezone**   | Handles timezone-aware date and time objects. |
| **timedelta**  | Performs date/time arithmetic. |
