
These are standard **data types** you can use in the Chatzy Sandbox environment.  
They define the kind of values a variable can hold.

| Type Name   | Description |
|-------------|-------------|
| **str**     | String type – text values. |
| **int**     | Integer type – whole numbers. |
| **float**   | Float type – decimal numbers. |
| **list**    | List type – an ordered collection of items. |
| **dict**    | Dictionary type – key-value pairs. |
| **set**     | Set type – an unordered collection of unique items. |
| **tuple**   | Tuple type – an ordered, unchangeable collection of items. |
