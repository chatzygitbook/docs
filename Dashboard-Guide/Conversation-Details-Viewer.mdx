Displays the full exchange between the user and the AI agent.

**Features:**

- **User Message**  
  Shows what the user said, in exact wording.

- **Agent Response**  
  Displays the reply given by your AI agent (whether flow-based or conversational).

- **Revise Answer**  
  Allows you to manually edit and update the agent’s response directly from the dashboard — useful for quality review or lead correction.

- **Expected Response (Optional)**  
  If you’ve configured expected answers, you can compare the actual vs. expected.

- **Uploaded Documents**  
  If the user uploaded any files (e.g., resumes, ID proofs), they will be shown here for preview/download.
