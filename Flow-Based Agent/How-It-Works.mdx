- **Nodes** are blocks of logic in your bot.
- A flow starts with a **Start Node**.
- Each node can contain multiple **cards**.
- Connect nodes using **connectors** to define sequence.

**Cards** define actions/inputs/outputs:
- Capture inputs
- Show messages
- Perform backend actions
- Use AI for summarization, classification, or generation
